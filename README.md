!\[Stocks](https://img.shields.io/badge/Stocks-15%20tickers-green)



An end-to-end MLOps pipeline that screens 15 major US stocks using machine learning

— generating \*\*BUY / HOLD / SELL\*\* signals from technical indicators with an interactive Streamlit dashboard.



\## ✨ Features



\- 📊 \*\*Screener\*\* — BUY/HOLD/SELL signals with confidence scores for 15 stocks

\- 📂 \*\*Custom CSV\*\* — Upload any stock CSV and get instant RSI, MACD, candlestick + ML signal

\- 📈 \*\*Stock Detail\*\* — Candlestick with MA20/MA50, RSI, MACD per ticker

\- 🤖 \*\*Train Models\*\* — Train 3 ML models directly from the dashboard
